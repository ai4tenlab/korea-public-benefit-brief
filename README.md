# 정책과 시민 | Korea Public Benefit Brief

정책이 시민의 삶에 닿는 순간을 공식 출처로 확인하고 설명하는 공익정책 GitHub Pages 브리프입니다.

## 핵심 원칙

- 정부·지자체·공공기관의 공식 원문 우선 확인
- 대상·기준·기한·신청 경로를 분리해 설명
- 과장·신청 성공 보장·정책자금 대행 표현 금지
- 정당·후보·이념을 지지하거나 반대하지 않는 비당파적 공익 편집
- 3줄 요약, 한 문장 답변, FAQ, 출처 블록을 통한 시민 친화적 정보 구조

## GitHub Pages

이 저장소는 GitHub Pages 배포를 전제로 한 Jekyll 정적 블로그입니다.
